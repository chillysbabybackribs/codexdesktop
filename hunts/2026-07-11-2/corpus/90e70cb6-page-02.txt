### Call from code with the Zapier SDK

Trigger Google Forms and AI by Zapier actions from backend services, scripts, or agent code.

Install

Run an action

import { createZapierSdk } from '@zapier/zapier-sdk';

const zapier = createZapierSdk();

await zapier.runAction({ /* google-forms / ai action */ });
