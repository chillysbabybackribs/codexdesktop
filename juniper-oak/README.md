# Juniper & Oak

A standalone one-page concept site for a neighborhood coffee shop.

## Run locally

```bash
cd juniper-oak
npm run dev
```

## Visual contract

- Direction: light editorial neighborhood café; tactile, warm, and quietly distinctive.
- Primary action: plan a visit, with hours and directions available on the same page.
- Typography: high-contrast system serif display paired with a compact humanist sans-serif UI voice.
- Palette: parchment canvas, deep evergreen surfaces, burnt-copper emphasis, ochre details.
- Composition: split first viewport with oversized live type and a dominant arched environmental photograph; featured drinks move from photo-plus-menu to a vertical mobile layout.
- Motion: short reveal transitions only; all nonessential movement is removed for reduced-motion users.

## Image provenance

- `assets/juniper-oak-hero.webp` — generated with the built-in OpenAI image tool for this project; 1536 × 1024. Responsive focal point stays near 64% horizontally. Replace at the same aspect ratio if real café photography becomes available.
- `assets/featured-drinks.webp` — generated with the built-in OpenAI image tool for this project; 1701 × 925. Cropped as a wide group image, with all three drinks retained at supported breakpoints.

Both images have fixed intrinsic dimensions, WebP compression, and meaningful alternative text in `index.html`. A later deployment can add responsive AVIF sources if its browser-support target allows it.
