One focused workspace

Keep the work in view.

A chat panel is only half the job. Codex Desktop puts the conversation next to the things the work actually touches — so context never leaves the window.

A real browser, right beside the chat

The browser pane is a native Chromium surface — not a screenshot, not an iframe. The agent can open pages, research, and drive the web while you watch it happen in the same window.

Beside your files

Point it at a workspace and it edits in place — diffs, file changes, and reviews stay attached to the thread.

Durable memory

Provider-neutral long-term memory carries context between conversations, so it recalls what you told it last week.

Local-first, your keys

Runs on your machine and talks to the providers you choose. Your workspace and credentials stay yours.

Codex & Claude

Switch the engine behind the same interface — Codex or Claude — without leaving the app you already have open.

Skills, discovered locally

Drop reusable workflows into your project and Codex Desktop finds and attaches them automatically — no registry round-trip, no copy-paste into a prompt.
