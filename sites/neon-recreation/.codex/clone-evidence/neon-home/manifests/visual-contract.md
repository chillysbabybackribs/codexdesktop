# Observable visual contract

- Canvas: near-total matte black with white primary text, cool gray secondary text, and occasional pale mint full-width sections.
- Header: 36px announcement plus 64px sticky navigation; mobile uses a 56px header and full-height black menu sheet.
- Hero: source-matched 1920/1304/1016px responsive video stage using Neon’s exact 9.375s media loop, bottom-weighted copy, 72px/1.12 desktop headline with tight negative tracking, two restrained pill actions, and five equal capability columns.
- Density: generous vertical pacing; most primary sections occupy 800–1200px and use one dominant idea.
- Geometry: square product surfaces and diagrams; pills reserved for actions and tab-like choices; 1px cool-gray borders; almost no shadow except the main product window.
- Typography: Inter-like low-weight display type; Geist Mono-style labels and technical metadata; left alignment except for deliberately centered product statements.
- Responsive behavior: desktop nav collapses to menu; hero copy shrinks without losing the two-action row; capability cards become horizontal overflow; complex three-column diagrams reflow vertically.
- Motion: exact Neon AV1/HEVC/VP9 hero loop with the source mobile still-image replacement; 200ms interface state transitions; local video pauses at frame zero under reduced-motion preference.
